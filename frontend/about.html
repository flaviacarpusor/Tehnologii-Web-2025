Despre aplicație

ReT – Resource Recommender este o platformă web care ajută utilizatorii să descopere resurse relevante pe baza intereselor și preferințelor personale.

Funcționalități principale:

  • Autentificare și gestionare cont utilizator
  • Filtrare și recomandare resurse
  • Gestionare preferințe personale
  • Schimbare parolă și recuperare parolă uitată
  • Interfață responsivă și ușor de folosit

Tehnologii folosite:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js
  • Bază de date: PostgreSQL
  • REST API
